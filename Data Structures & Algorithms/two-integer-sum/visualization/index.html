Algorithm Visualization

Goal: Find the indices of two numbers in an array that add up to a specific target.

Approach (Hash Map):

  • Initialize an empty Hash Map (dictionary) called store. It will map number -> index.
  • Iterate through the array. For each number at index i:
  • Calculate the complement you need: rem = target - nums[i].
  • Check if this `rem` already exists in your store.
  • If it does (and it's not the exact same index), we've found our pair! Return [store[rem], i].
  • If not, save the current number and its index in the store so future numbers can find it.
  • This enables a highly efficient single pass (O(N) time complexity).

Python Code

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        store = {}

        for i in range(len(nums)):
            rem = target - nums[i]

            if rem in store and store[rem] != i:
                return [store[rem], i]

            store[nums[i]] = i
