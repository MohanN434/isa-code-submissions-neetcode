Algorithm Visualization

Goal: Find the longest prefix that is common among all strings in an array.

Approach:

  • Take the first string as the initial "longest common prefix" hypothesis.
  • Iterate through the remaining strings in the array.
  • For each string, match its characters against the current prefix one by one.
  • Stop comparing as soon as a mismatch is found or you reach the end of either string.
  • Update the hypothesized longest common prefix to just the consistently matched characters.
  • If the prefix becomes completely empty at any point, we can return "" early.

class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        longest = strs[0]

        for s in strs:
            c_long = ""
            i = 0

            while i < len(s) and i < len(longest):
                if s[i] != longest[i]:
                    break
                c_long += s[i]
                i += 1
            
            longest = c_long
            if longest == "":
                return ""

        return longest
