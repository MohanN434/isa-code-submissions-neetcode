Algorithm Visualization

Goal: Remove all occurrences of "val" in-place and return the final size of the array.

Approach (Two Pointers):

  • Use a loop index variable i to iterate through every element in the array.
  • Use another variable idx to track where the next safe element should be placed.
  • If the current element nums[i] is NOT the value we want to remove, swap it into the safe position nums[idx].
  • Increment idx to mark the next safe position.
  • If the element IS the value we want to remove, we simply skip iterating over it, advancing only i.
  • Finally, returning idx yields the number of valid elements left in the list.

Python Code

class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        idx = 0

        for i in range(len(nums)):
            if nums[i] != val:
                nums[i], nums[idx] = nums[idx], nums[i]
                idx += 1

        return idx
